TODAY'S BGM

プライバシーポリシー

制定日: 2026年7月23日

「今日の作業テーマ曲」(以下「本アプリ」)は、個人が非商用で開発・公開している作業用BGMレコメンドWebアプリです。 本アプリは利用者の個人情報を収集するサーバーを持たず、あなたのデータはあなたの端末(と、任意で連携した場合のみ、あなた自身のGoogleドライブ)にだけ保存されます。

1. 収集するデータ — ありません

本アプリは静的なWebページのみで構成されており、開発者が運営するサーバーやデータベースは存在しません。利用者の氏名・メールアドレス・利用状況などを開発者が収集・保存・閲覧することはありません。

  • アクセス解析ツール(Google Analytics 等)は使用していません
  • 本アプリ自身がCookieを発行することはありません
  • 広告は表示しません(YouTube動画内の広告はYouTube側によるものです)

2. 端末内に保存されるデータ(localStorage)

本アプリの動作に必要な次のデータは、すべてお使いのブラウザの localStorage(端末内)にのみ保存されます。開発者を含む第三者に送信されることはありません。

  • おすすめの条件(好みのジャンル・時間などの設定)
  • 曲への★評価
  • 再生履歴・提示履歴と、その表示に使う曲情報の控え
  • 再生方法などのアプリ設定

これらはアプリ内の「設定 → データをすべてリセット」やブラウザのサイトデータ削除でいつでも消去できます。

3. データ同期(任意・Googleアカウント連携)

複数端末で評価・履歴・設定を引き継ぐため、任意でGoogleアカウントと連携できます。連携しなくても本アプリのすべての基本機能を利用できます。

連携した場合、上記のデータはあなた自身のGoogleドライブ内のアプリ専用領域(appDataFolder)にのみ保存されます。

  • 使用するスコープは https://www.googleapis.com/auth/drive.appdata のみです。あなたのGoogleドライブ内の通常のファイルや他のデータに本アプリがアクセスすることはできません
  • 保存先はあなたのGoogleアカウントの領域であり、開発者がその内容にアクセスする手段はありません
  • 同期データを開発者や第三者に送信・販売・共有することはありません

本アプリによるGoogle APIから取得した情報の利用は、Google API Services User Data Policy(Limited Use の要件を含む)に準拠します。

4. 外部サービスとの通信

本アプリは動作のために次の外部サービスと通信します。その際、通信先の事業者(Google)が接続元情報(IPアドレス等)を受け取り、各社のポリシーに基づいてCookie等を利用する場合があります。

  • YouTube 埋め込みプレイヤー・サムネイル画像: 曲の再生に使用します。埋め込みプレイヤーの利用には YouTube 利用規約 が適用されます
  • Google Fonts: 画面表示用フォントの読み込みに使用します
  • Google Identity Services / Google Drive API: データ同期を有効にした場合のみ使用します

Googleによる情報の取り扱いは Google プライバシーポリシー をご確認ください。

5. データの削除・連携の解除

  • 端末内のデータ: アプリ内「設定 → データ」の各リセット(評価のみ・履歴のみ・すべて)で消去できます
  • Googleアカウント連携の解除: Googleアカウントの「サードパーティ製のアプリとサービス」から本アプリのアクセス権を削除できます
  • Googleドライブ上の同期データの削除: Googleドライブの「設定 → アプリの管理」から本アプリの非表示のアプリデータを削除できます

6. 免責事項

  • 本アプリで再生される楽曲・動画はYouTube上のコンテンツであり、その権利は各権利者に帰属します。本アプリは動画ファイルを保存・再配布しません
  • 動画の公開状況・再生可否・広告の表示はYouTube側の事情により変化します。特定の曲が再生できることを保証するものではありません
  • 本アプリの利用により生じたいかなる損害についても、開発者は責任を負いかねます

7. ポリシーの改定

本ポリシーの内容は、機能の追加・変更に応じて改定することがあります。重要な変更がある場合は本ページで告知し、冒頭の日付を更新します。
